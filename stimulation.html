Instructions to enter plain-text:

  • Default bit-stream is specified but you can modify it and enter your custom bit stream.
  • The max size of input bit-stream is 16-bit.

    • Plaintext:


    Instructions to enter key-value:

  • Default key value is specified but you can modify it and enter your custom value stream.
  • The max size of key value is 16-bit.
  • The length of key value should be equal to the input plain-text bits.

    • key:

    To generate the cipher-text press encrypt button.

    Ciphertext:


    Instructions to obtain plain-text from the cipher text:

  • Remove the plain-text value from the input field and press the decrypt button.
  • The resulting plain-text after decryption will be generated in the input field.
